MusEmbed

by Paraborg Discord Bots

"A Discord bot that helps you create embeds while playing you music."


This is a bot that can help you easily create embeds in your server. We also have music playing functionality.

Customizable prefix. Default is `em/`.


General Commands:
`help`: Shows this help message.
`botinfo`: Retrieves information about the bot.
`serverinfo`: Retrieves information about the server.
`ping`: Shows latency of the bot and the API.
`vote`: Links you to vote for our bot.

Embed Commands:
`embed [message]`: Embeds your message.
`rawembed [message]`: Administrator-only command that embeds your message without your name.

Music Commands:
`play [name of music]`: Searches for the song you requested.
`skip`: Skips the current song.
`np` Tells you what song is playing.
`volume ([number])`: Sets the volume. Checks the volume if you don't provide a number.
`queue`: Lists the queue.

Moderation Commands:
`kick [user]`: Kicks a user from the server.
`ban [user]`: Bans a user from the executed server.
`purge [number]`: Deletes a number of messages in a channel.
`mute [user]`: Mutes a user in the executed server. A 'Muted' role must first be set up for this to work.
`unmute [user]`: Unmutes a user in the executed server.

Server Configurations:
`showconf`: Shows the configurations for your server.
`setconf [item] [new value]`: Sets a new value for your server's configuration.
(Available configurations: `Prefix`, `Censor` `on/off`)
